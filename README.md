## Introduction
This guide covers tools that every student should know about.
Each section is written by a different team member.

## VS Code
Supports many languages: Python, JavaScript, C++, HTML/CSS, and more.
IntelliSense: Smart code completion and suggestions help you learn syntax as you go
Extensions marketplace: Add tools for debugging, formatting, version control, and learning resources.
Integrated terminal: Run code without switching apps.
Git integration: Track changes and collaborate on code

## Notion
Class notes + lecture organization: Create pages per course, embed files, audio, and links.
Study trackers & calendars: Build to-dos, deadlines, and revision schedules.
Custom templates: Use or build templates for lecture summaries, lab reports, or exam planners.
Databases: Track assignments, reading lists, habits, and grades in sortable tables.
## Canva
Presentations with impact: Create stylish slides fast.
Infographics & posters: Great for reports, projects, and class displays.
Templates galore: Access thousands of ready-made designs.
Team collaboration: Work on designs with classmates.
